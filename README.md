# 42 School Projects

This repository contains projects developed as part of the [42](https://42.fr) curriculum — a peer-to-peer, project-based programming school with no teachers or traditional classes. Students learn by doing, submitting projects that are evaluated by peers using a strict norm and automated graders.

All code is written in C and must comply with the **42 Norm**: a strict coding standard that enforces formatting, function length, variable declarations, and more.

---

## Milestone 00

### [libft](milestone_00/libft/)

A from-scratch reimplementation of a subset of the C standard library, plus additional utility functions. This is the foundation project — the resulting library is reused in almost every subsequent project.

---

## Milestone 01

### [get_next_line](milestone_01/get_next_line/)

Reads a single line from a file descriptor each time it is called, handling any buffer size and multiple file descriptors simultaneously. Teaches static variables and memory management across calls.

### [ft_printf](milestone_01/printf/)

A partial reimplementation of `printf`, supporting the main conversion specifiers (`%c`, `%s`, `%d`, `%i`, `%u`, `%x`, `%X`, `%p`, `%%`). Teaches variadic functions and output formatting.
